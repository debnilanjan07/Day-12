Position Absolute

- It will always relaive to its parent element if our parent element is not present it will go for root element.

Position Relative

- It is always related to its next element or related to its sibling.

Position Fixed

Fixed - The property fixed is relative to the viewport , when we apply position fixed property it gets fixed to that position so it doesn't move when we scroll the page up or down. It doesn't takes the additional space of the page and doesn't disturb other elements.
